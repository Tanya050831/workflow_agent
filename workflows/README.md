## Workflow Overview | 工作流概览

Our physics problem-solving agent is implemented on the Coze platform. The workflow consists of several key components:

我们的物理问题求解智能体在Coze平台上实现。工作流包含以下几个关键组件：

### Key Components | 关键组件

1. **Input Processing | 输入处理**
   - Receives multimodal input (text and diagrams)
   - 接收多模态输入（文本和图表）

2. **Language Translation | 语言翻译**
   - Chinese-to-English translation preprocessing
   - 中英文翻译预处理

3. **RAG (Retrieval-Augmented Generation) | 检索增强生成**
   - Textbook knowledge retrieval
   - 教科书知识检索
   - Solved problems database retrieval
   - 已解决问题数据库检索

4. **Core Reasoning Model | 核心推理模型**
   - Model_Deepseek-r1 for reasoning
   - 使用Model_Deepseek-r1进行推理

5. **Feedback Loop | 反馈循环**
   - Self-correcting mechanism
   - 自我纠正机制

6. **Output Generation | 输出生成**
   - Generates final solution
   - 生成最终解答
